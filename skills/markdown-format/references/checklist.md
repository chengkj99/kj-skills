# Markdown 格式化自检清单

## 结构

- [ ] YAML frontmatter 完整、未被包进代码块
- [ ] 章 `#`、节 `##`、小节 `###` 层级连续，无跳级
- [ ] 无仅用 `**粗体**` 充当标题的长段落

## 表格

- [ ] 每个 GFM 表：行 1 表头 → 行 2 `| --- |`（列数匹配）→ 行 3+ 数据
- [ ] 无 `|  |  |` 空表头残留
- [ ] 单元格内管道符已转义或改写，不破坏列对齐
- [ ] Obsidian 预览中表格正常渲染

## 代码

- [ ] 围栏语言正确（bash / json / markdown / yaml / text）
- [ ] JSON 2 空格缩进；字符串内引号转义正确
- [ ] 无逐行 `>` 伪代码块残留
- [ ] Pandoc 反斜杠转义已清理（`\|`、`\_`、`\[说明\]` → 正常符号）

## 版式

- [ ] 列表项之间无多余空行（引用块内列表同理）
- [ ] 有序列表编号连续、与章节逻辑一致
- [ ] 提示框统一为 `> **标题**` + 正文
- [ ] 代码块前后各留一空行（便于渲染）

## 内容与合规

- [ ] 未删除用户未确认的段落、图片、附录
- [ ] 相对路径图片、wikilink 仍有效
- [ ] 若改 `raw/`：用户已明确授权
