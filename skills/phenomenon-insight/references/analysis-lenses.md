# Analysis Lenses

按需读取。用于把一个现象从“有意思”推进到“说透”。

## 第一性原理镜头

适合问题：

- “我是不是默认相信了什么？”
- “为什么这个结论听起来对，但执行时不对？”
- “大家都这么说，是否只是旧经验？”

问题清单：

1. 这个现象背后的默认前提是什么？
2. 这个前提来自事实、经验，还是群体共识？
3. 如果这个前提不成立，结论会变成什么？
4. 有哪些条件变化，让过去的经验失效了？

## 平台机制镜头

适合短视频、公众号、社群、AI 工具和推荐系统相关现象。

问题清单：

1. 平台奖励什么行为？
2. 平台过滤什么行为？
3. 哪些指标被放大，哪些信号被隐藏？
4. 用户看到的是内容本身，还是平台筛选后的结果？
5. 这个机制让创作者产生了什么误判？

## 人性机制镜头

适合解释熟人不点赞、用户不付费、读者不行动、同事不支持等现象。

常见变量：

- 身份成本：做出反应会不会暴露自己的态度。
- 比较心理：支持别人是否意味着承认自己落后。
- 行动成本：点赞、评论、转发、购买各自有不同心理门槛。
- 损失厌恶：人更怕失去面子、钱、确定性。
- 社交距离：陌生人更容易就事论事，熟人更容易带入关系判断。

## 内容传播镜头

适合把洞察转成选题。

判断标准：

1. 是否有读者熟悉的具体场景？
2. 是否能推翻一个常见误解？
3. 是否能用一句话转述？
4. 是否有行动建议，而不是只讲道理？
5. 是否有下一集可接？

## 反例镜头

每个洞见都要找边界。

追问：

1. 什么情况下这个解释不成立？
2. 有没有更简单的解释？
3. 有没有数据或案例会推翻它？
4. 如果有人反驳，他最可能抓住哪一点？
5. 我是否把单个经历过度上升成规律？
