# web-slide · 网页演讲稿生成 Skill

基于 `skills-deck.html` 框架，AI 将用户大纲自动组装为**单文件 HTML 演讲稿**，继承完整交互能力。

---

## 快速使用

```
/web-slide 主题：Claude Code 实战入门，受众：技术团队，15张
```

或提供完整大纲：

```
帮我做一个技术分享，主题是「如何用 AI 提升开发效率」，
分 4 章：认知、方法、案例、总结，大约 18 张幻灯片
```

---

## 框架能力（单文件 HTML，双击即打开）

| 能力 | 操作 | 说明 |
|------|------|------|
| 键盘翻页 | `←` `→` 或空格 | 支持触屏滑动 |
| 演讲者备注 | `N` 键开关 | 从 `data-notes` 读取 |
| 全局索引 | `ESC` | 缩略图跳转 |
| 代码高亮 | 自动 | Prism.js |
| Mermaid 流程图 | 翻到该页自动渲染 | 支持多种图类型 |
| PDF 导出 | 点击「PDF」按钮 | html2canvas + jsPDF |
| Logo 自动切换 | 无需配置 | 深色页白logo，浅色页深logo |
| 进度条 | 自动 | 底部显示 N/总数 |

### 5. 章节幕（全景背景图变体）

最视觉震撼的章节页。需提供全景图片（建议 1920×1080）。

```css
/* 在 PRE_CONTENT 的 </style> 前追加 */
#s-epilogue.slide.chapter {
  background: url('your-image.png') center right / cover no-repeat, var(--ink);
}
#s-epilogue.slide.chapter::before {
  content: '';
  position: absolute;
  inset: 0;
  background: linear-gradient(to right, rgba(5,12,28,0.82) 0%, transparent 100%);
  pointer-events: none;
  z-index: 0;
}
```

适用：演讲收束/升华页，图片类型：宇宙、城市夜景、自然壮景

## 功能详解

### Logo 自动深/浅切换

框架 JS 监听幻灯片切换，自动判断背景深浅并切换 Logo：

| 幻灯片类型 | Logo 版本 |
|-----------|---------|
| `cover`（封面）| 白色版本 `chanyanxueyuan_white.png` |
| `chapter`（章节幕）| 白色版本 |
| `s37`、`s-qa`（特殊页）| 白色版本 |
| 其余内容页 | 深色版本 `chanyanxueyuan.png` |

**替换为自己的 logo：** 将自己的 logo 命名为 `chanyanxueyuan.png`（深色）和 `chanyanxueyuan_white.png`（白色），放在同目录即可。或修改框架中 `BRAND_LOGO_LIGHT` 和 `BRAND_LOGO_DARK` 变量。

---

### PDF 导出

点击导航栏「PDF」按钮，功能由 `html2canvas + jsPDF` 实现：

- 输出分辨率：1280×720 px（16:9）
- 格式：JPEG，质量 0.92
- 每页 Logo 会根据深浅自动注入
- Mermaid 图表会先转为 PNG 再截图（避免 SVG 渲染问题）
- 保存文件名：由生成时 `pdf.save('xxx.pdf')` 指定

**注意**：PDF 导出需要联网（CDN 加载 jsPDF），离线环境需预先下载 lib。

---

### 全景背景图章节幕

通过给特定 slide id 添加 CSS 背景覆盖实现。需要：
1. 图片文件（推荐 1920×1080，JPG 60-80 质量）
2. 在 PRE_CONTENT 的 `</style>` 结尾前追加 CSS

```css
#s-epilogue.slide.chapter {
  background:
    url('panorama.jpg') center right / cover no-repeat,
    var(--ink);
}
#s-epilogue.slide.chapter::before {
  content: '';
  position: absolute;
  inset: 0;
  background: linear-gradient(to right, rgba(5,12,28,0.82) 0%, transparent 100%);
  pointer-events: none;
  z-index: 0;
}
```

---

### data-build 渐进展示

在 `<section class="slide">` 上添加 `data-build="N"` 属性，该页每次按翻页键只显示一个 block（不跳到下一页），共需按 N+1 次才会翻到下一页。

```html
<section class="slide" id="s07" data-build="2" data-build-step="0" data-notes="…">
  <section class="s7-block"><!-- 第1次显示 --></section>
  <section class="s7-block"><!-- 第2次显示 --></section>
</section>
```

适用：需要停顿互动的问答、揭晓答案场景。

## 文件结构

```
skills-deck.html（框架源文件）
├── 前段（行 1–4812）
│   ├── head + CDN 引用（Mermaid/Prism/jsPDF/html2canvas）

│   │   ├── 各 slide 专属 CSS（#s01、#s04、#s06b…）
│   │   └── 导出/打印（@media print）
│   └── body + deck 容器 + brand logo
├── 内容区（行 4813–7333）← 生成时用你的 <section> 替换
└── 后段（行 7334–8468）
    ├── 导航栏（← → 按钮 + PDF 按钮）
    ├── 进度条
    ├── 演讲者备注面板
    ├── 索引模式
    ├── Prism / html2canvas / jsPDF（CDN script）
    └── 主脚本（翻页、备注、索引、Logo切换、PDF导出）
```
