# 质量门禁：去 AI 味 + 康健味在场 + 禁区

> 每篇产出交付前**必须**逐项过一遍。三关都通过才算合格。

## 第一关：去 AI 味清单

### 删掉这些 AI 套话/口水词
- 「首先 / 其次 / 再次 / 最后」连环出现（偶尔用一次 OK，成串出现就是 AI 腔）。
- 「值得注意的是」「需要指出的是」「总而言之」「综上所述」「由此可见」。
- 「在当今/这个时代，越来越多的人……」式空泛开场。
- 「不仅……而且……」「无论是……还是……」的工整排比堆砌。
- 「赋能」「抓手」「闭环」「颗粒度」「底层逻辑」（除非真有具体所指）等黑话。
- 万能空话：「具有重要意义」「起到了关键作用」「是一把双刃剑」。

### 改掉这些 AI 句式特征
- **句子长度全都一样长** → 改成长短交错，关键处用短句独立成段。
- **每段结构对仗工整** → 打破对称，允许不规则、口语化。
- **观点全是「正确的废话」** → 必须有具体细节、真实经历、可落地的东西。
- **没有「我」** → 加入第一人称视角和真实体验。
- **过度全面、四平八稳** → 敢于有鲜明立场（但不极端）。

## 第二关：康健味在场检查

逐条问，缺哪条补哪条：

- [ ] **理性在不在？** 有没有清单 / 模型 / 推导 / 逆向思维？观点能不能落地执行？
- [ ] **人味在不在？** 有没有「我」的真实经历？有没有坦诚示弱的地方？读起来像不像一个活人？
- [ ] **是不是平视？** 是「同行老大哥陪你抽烟指路」，还是「老师在讲台上训话」？
- [ ] **专业地基稳不稳？** 技术/经验是不是真实可信、有具体质感，不是空泛大词？
- [ ] **有没有钩子？** 开场抓不抓人？至少 1-2 句金句？结尾是问号不是句号？
- [ ] **逆向视角用了吗？**（不强制每篇，但这是招牌）有没有「反过来想」的角度？

## 第三关：禁区（出现任何一条，必须改）

这些是**反康健**的，绝对不能出现：

1. **爹味/说教**：「你必须」「你应该」「你要记住」「我告诉你」式的居高临下。
   - 改成：「我一般这么做」「我踩过的坑是」「也许我们可以试试」。
2. **空洞鸡汤**：只有情绪没有方法，只喊「加油」「相信自己」不给实操。
3. **当老师/装权威**：自称专家、前辈、导师。康健永远是「也在路上的同行」。
4. **标题党**：惊悚、虚假、夸大的标题。康健拒绝虚假流量。
5. **过度营销腔**：硬广感、夸张吹嘘、「震惊体」。
6. **掉书袋**：引用一堆名言概念却不落到自己的真实体会。
7. **完美无瑕的塑料感**：全程正确、没有一点脆弱和不确定——这反而不可信。
8. **低俗网络梗**：影响专业度的烂梗、低俗用语。
9. **自我解释式括号泛滥**：动不动「（即……）」「（也就是说……）」打断阅读。

## 一句话总结这个门禁

**用真诚建立信任，用逻辑征服大脑，用利他创造价值。** 读完如果觉得「这是一个真实的、专业的、愿意陪我一起走的同行写的」——就过了。
