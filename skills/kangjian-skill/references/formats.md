## 一、公众号文章（主力）

康健的主力形态。深度 + 个人观点，要让读者「获得」点什么（认知升级或行动指南）。

### 标准结构：起承转合 + 落地

1. **引入 Hook**：一个具体的故事 / 对话 / 生活场景 / 反常识提问开场。
   - ✅「昨天接到一个推销电话……」「停更这半年，我其实在……」
   - ❌「本文将讨论……」「在当今时代，越来越多的人……」
2. **展开 Body**：逻辑严密地讲观点，结合个人经历或权威引用（书摘/名言）。用 H2/H3 切分，列表化。
3. **升华 Climax**：从具体事件提炼出普适的认知模型或哲理（以小见大）。
4. **落地 Action**：给具体的行动清单 / 思维模型 / SOP / Checklist，让人想收藏。
5. **金句收尾 Punchline**：一句有力度的金句 + 一个互动钩子（二选一站队 / 经历征集）。

### 格式规范
- H2/H3 分层，每段 3-5 行，多留白。
- 关键句、结论句用 **加粗**。
- 概念用直角引号「」。
- 全文至少 1-2 句可截图分享的金句。
- 标题：拒绝标题党，用「痛点 + 反差」（如《如何过上痛苦的生活》《两次拒绝了20万》）。

### 篇幅
正文 1500-3000 字为宜。深度够、但不注水。

## 二、短视频口播脚本

口语化、钩子驱动，写出来的是「能直接对着摄像头念」的话，不是「书面文章」。

### 标准结构

1. **3 秒钩子**（开头第一句必须抓人）：
   - 反常识结论 / 一个具体冲突场景 / 直接点破读者痛点。
   - ✅「我两次拒绝了银行白给的 20 万，今天说说为什么。」
   - ❌「大家好，今天我想跟大家聊一个话题。」
2. **冲突/铺垫**：用一个具体的场景或自己的经历把人带进来。
3. **核心观点**：1 个主观点讲透，不贪多。口播一条片子只讲一件事。
4. **落地/反转**：给一个能立刻用的结论或一个认知反转。
5. **收尾钩子**：金句 + 引导互动（评论区站队 / 关注同行）。

### 口播语感规则
- **短句、口语、能换气**。一句话别超过一口气能念完的长度。
- 删掉所有书面连接词（「综上所述」「由此可见」），换成口语（「所以你看」「说白了」「这事儿吧」）。
- 多用「你」直接对着观众说，拉近距离。
- 适当留口头禅和停顿感，别太「完美」——康健本人就承认自己「节奏慢、有口头禅」，这是活人感。
- 标注【停顿】【强调】等口播提示（可选）。

### 篇幅
按时长，60 秒约 200-260 字，90 秒约 300-380 字。开头 3 秒最关键。

## 三、AI 编程教程

专业 + 可读性并重。康健的可信度来自真实的 10 年代码经验，所以**技术必须准确**，同时**说人话**。

### 核心原则：知其然，也知其所以然
- 不只给「怎么做」，还要讲「为什么这么做」「不这么做会踩什么坑」。
- **AI-first 但不偷懒**：演示用 AI 高效完成的同时，必要时保留「手动版/原理版」，让读者理解底层，而不是只会复制粘贴。

### 标准结构

1. **痛点引入**：用一个真实开发场景说明「为什么需要这个」。不要上来就讲 API。
2. **目标**：学完这节能做到什么（具体、可验证）。
3. **分步骤实操**：每步有明确动作 + 预期结果。代码块完整可运行。
4. **原理/避坑**：关键处补一句「这背后是因为……」或「这里很多人会踩……」。
5. **小结 + 下一步**：总结要点，给「下一讲/下一步」的衔接（多章时务必检查衔接连续、不串台）。

### 语感规则
- 专业术语准确，但首次出现时用一句大白话解释。
- 可以有轻松的幽默，但不能牺牲准确性。
- 避免爹味：不说「你必须」「你应该」，多说「我一般这么做」「我踩过的坑是」。

## 四、演讲稿

现场感、节奏感。写的是「站在台上说出来」的稿子，要照顾听众的注意力曲线。

### 标准结构

1. **开场抓注意**：一个故事、一个提问、一个反差数据。建立「我和你们是一类人」的同场感。
2. **抛核心命题**：今天就讲一件事，一句话说清。
3. **主体（2-4 个段落）**：每段一个分论点，配一个故事或案例。段落之间有明确过渡句。
4. **情绪高潮**：一个升华时刻，常用康健的「顿悟」叙事（「就在那一瞬间，我明白了」）。
5. **收尾**：一句金句 + 一个对听众的邀请/号召（「我们一起……」）。

### 演讲语感规则
- **句子比文章更短**，因为是说出来的。
- **留白与停顿**：标注【停顿】【看向观众】等可选提示。
- **重复与排比**强化记忆点（重要句子可重复一遍）。
- **有节奏起伏**：紧张—舒缓—高潮，别一条直线。
- 平视听众，绝不爹味说教，永远「同行 + 带路人」的姿态。
