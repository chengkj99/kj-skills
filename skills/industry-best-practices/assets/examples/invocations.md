# Invocation Examples｜调用示例

## 通用产品调研

```text
请使用 $industry-best-practices，调研“B2B SaaS 客服系统的自助服务体验最佳实践”。

背景：我们做企业客服系统。
目标：找到最近 30 天值得借鉴的官方能力变化、论文、开源项目、工程实践、专家观点、研究机构材料和竞品动作。
输出：结论、信号地图、优化建议池、MVP、指标和风险。
```

## 竞品扫描

```text
请使用 $industry-best-practices，调研“海外主流项目管理工具在自动化工作流上的最新做法”。

关注：能力边界、交互设计、定价策略、用户反馈、工程实现线索。
输出：竞品信号地图、可借鉴实践、不可直接照搬的风险、1 个月内可验证的 MVP。
```

## 技术方案最佳实践

```text
请使用 $industry-best-practices，调研“实时数据看板的性能与可靠性最佳实践”。

背景：我们有一个内部运营看板，数据延迟和加载速度不稳定。
输出：工程实践、架构建议、监控指标、灰度验证方案和风险。
```

## 每日/每周推送

```text
请使用 $industry-best-practices，生成本周“企业知识管理产品”的行业最佳实践周报。

时间范围：最近 7 天。
输出：Top 信号、趋势变化、机会池、风险噪音、下周深挖问题。
```
