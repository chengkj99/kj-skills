# Daily / Weekly Automation Prompt

## 每日行业最佳实践

```text
请使用 $industry-best-practices 生成今天的行业最佳实践日报。

主题：{topic}
优化对象：{product_or_capability}
时间范围：最近 24 小时，必要时补充最近 7 天背景。
关注来源：官方发布、论文、开源项目、工程社区、行业专家、研究机构、竞品动态、市场反馈。
时效性策略：按材料类型和领域变化速度动态计权；旧标准/经典方法可保留但必须说明当前有效性；超过 1 年且无近期复核或采用证据的内容降权。

输出：
1. 今日 Top 5 信号
2. 今日最重要范式变化
3. 今日最值得看的研究/方法
4. 今日最值得看的开源或工程实践
5. 今日最值得看的竞品动作
6. 进入机会池的 3 条优化建议
7. 噪音/风险
8. 明天或本周最值得深挖的问题

要求：每条信号必须包含来源、时间、材料类型、有效新鲜度、时效判断、证据等级、产品启发和是否进入机会池。
```

## 每周机会池复盘

```text
请使用 $industry-best-practices 生成本周调研机会池复盘。

主题：{topic}
时间范围：最近 7 天。
目标：从本周所有信号中筛选最值得进入产品路线图、技术路线图或流程改进计划的机会。

输出：
1. 本周趋势总结
2. 本周高分信号 Top 10
3. 进入需求池的机会 Top 5
4. 应该忽略的噪音 Top 5
5. 建议下周深挖方向
6. 1 周/1 月/1 季度行动计划
```
