# Content Growth Review Framework

## Diagnostic Questions

Use these questions during every review:

1. What content produced saves, comments, shares, or conversion signals?
2. Which strong videos lack a clear follow-up series?
3. Which comments reveal questions, objections, or buyer intent?
4. Is the account promise obvious from the latest 10-20 topics?
5. Which topics attract the wrong audience or only create low-conversion debate?
6. What should be repeated, reframed, paused, or turned into a series?

## Topic Scoring

Score candidate topics 1-5:

- Audience pain: how urgent and concrete is the problem?
- Proof: does existing data/comment/transcript support it?
- Positioning fit: does it strengthen the account promise?
- Series fit: can it lead to a next episode?
- Conversion fit: does it naturally connect to资料包、群聊、课程、产品?

Prioritize topics with total score >= 20.
